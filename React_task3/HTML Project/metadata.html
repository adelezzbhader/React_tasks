Metadata Elements

<meta charset="UTF-8"> specifies the character encoding for the HTML document.

<meta name="viewport" content="width=device-width, initial-scale=1.0"> ensures proper rendering and touch zooming on mobile devices.

<title> defines the title of the document, shown in the browser's title bar or tab.

<link rel="stylesheet" href="styles.css"> links to an external CSS file.
