Images

<img> defines an image. It requires the src attribute to specify the image source and the alt attribute to provide alternative text.

Audio

<audio> defines an audio file. The controls attribute adds audio controls like play, pause, and volume.

Video

<video> defines a video file. It requires the width, height, and controls attributes.
